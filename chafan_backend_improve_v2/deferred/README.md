# Deferred Proposals

Work captured here is **not part of v2**. Each file documents:
- What the proposal would do.
- Why it's deferred.
- What would trigger revisiting.

These are not "next up" items. They are "documented so we don't lose the design work, revisit only when a concrete reason appears."

## Review cadence

These files are not on a calendar review. They are reviewed **when v2 ships**, plus whenever a per-file trigger fires. Until then, they sit dormant by design. A file lingering here is not a sign of rot; it's the intended state. Same pattern as `../phase5-optional/`.
